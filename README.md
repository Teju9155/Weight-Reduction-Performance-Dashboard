# Weight Reduction Performance Dashboard

## Overview

An interactive Power BI dashboard developed to monitor weight reduction progress using KPI tracking, trend analysis, and data visualization.

---

## Dashboard Features

- Starting Weight
- Current Weight
- Target Weight
- Weight Lost
- Goal Achievement %
- Program Status
- Weight Trend Analysis
- Correlation Analysis
- Lead Indicator Performance
- Root Cause Analysis
- Corrective & Preventive Action Tracker

---

## Tools Used

- Power BI
- Power Query
- DAX
- Data Modeling
- Data Visualization

## Skills Demonstrated

- KPI Reporting
- Dashboard Development
